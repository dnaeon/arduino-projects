## Arduino IR decoder

Decode your IR remote control codes using this Arduino sketch

## Components

* Arduino Uno
* Breadboard
* [IR Receiver Diode - TSOP38238](https://www.sparkfun.com/products/10266)

## Libraries

* [IRremote](https://github.com/shirriff/Arduino-IRremote)
