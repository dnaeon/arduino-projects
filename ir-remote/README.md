## Arduino IR remote control

Arduino IR remote control for controlling my TV at home :)

## Components

* Arduino Uno
* Breadboard
* [IR LED](https://www.sparkfun.com/products/9349)

## Libraries

* [IRremote](https://github.com/shirriff/Arduino-IRremote)
